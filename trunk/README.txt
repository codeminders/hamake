Hadoop "Make" utility
=====================

Please visit http://code.google.com/p/hamake/ for more information,
which includes documentaition, wiki, discussion groups, etc.


Requirements:

* Hadoop-0.18.3
* Python 2.5
* Pig-0.2.0 (optional)
* Hadoop Thrift Server (from Hadoop-0.19 contrib, backported to 0.18.3)

Installation:

python setup.py install
